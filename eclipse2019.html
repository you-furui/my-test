開発環境としてEclipse 2021以前を使用する

WindowsにPleiades All in OneをインストールしてEclipseを使用する方法について説明します。

32ビット版WindowsでEclipse 2018-09 32ビット版を使用する場合にも本ページを参考にしてください。

なお、本ページの手順は次の各バージョンでテスト済みです。

  • Windows 10 Home (64 bit) + Eclipse 2021-12, Eclipse 2021-09, Eclipse 2021-06, Eclipse 2021-03, Eclipse 2020-12, Eclipse 2020-09, Eclipse 2020-06, Eclipse 2020-03, Eclipse 2019-12, Eclipse 2019-06
  • Windows 7 Home Premium (32 bit) + Eclipse 2018-09 (Windows 32bit版)
    ※Win 10の32ビット版が手元に無かったので、たまたま残っていたWin 7を使用しました。

Step-1. 便利なツールのインストール

7-Zipは圧縮・展開ツールです。

  • 7-Zipのインストール (PowerPoint版, PDF版)

WindowsにEclipse 2021-12までのバージョンをインストールするときには7-Zipが必要です。 （すでに7-Zipがインストール済みであれば、このStep-1の作業は必要ありません）

Step-2. Eclipseのインストール

この手順は本書2.1節および2.4節前半に記載されているもののアップデート版です。

  • Pleiades All in One Eclipse 2021以前のインストール (PowerPoint版, PDF版)

インストールにはディスクに十分な空き容量が必要です。 インストール完了後は、ダウンロードした圧縮ファイルを削除しても差し支えありません。

Step-3. Eclipseの操作

この手順は本書2.2節および2.4節後半に記載されているものとほぼ同じです。

  • Eclipseの操作 (PowerPoint版, PDF版)
